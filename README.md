#  Number Guessing Game (Java)

## Description
This is a Java console application where the computer generates a random number between 1 and 100, and the user has to guess it within 10 attempts.

## Features
- Random number generation
- Maximum 10 attempts
- Too High / Too Low hints
- Score calculation
- Play Again option

## Technologies Used
- Java
- VS Code
- JDK 24

## How to Run

Compile:
```bash
javac NumberGame.java
```

Run:
```bash
java NumberGame
```
